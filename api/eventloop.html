must_the_show_go_on()[source]¶

Returns True if the program should continue to run, False otherwise.

The program can be shut down either via a regular shutdown (all processing stops immediately, might leave unfinished jobs), or a graceful shutdown, which will stop receiving events, finish processing all jobs, and then shutdown.

next_event_queue_item()[source]¶

! @brief Generator of event queue items, for sequential processing. @returns eva.eventqueue.EventQueueItem

This generator is needed in order to quickly iterate the main loop. If a lot of events are in the event queue, it may take too much time to run through them all, resulting in a timeout at the Kafka queue.
