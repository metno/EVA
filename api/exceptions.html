eva.exceptions¶

Exceptions thrown by EVA under various circumstances.

exception eva.exceptions.AlreadyRunningException[source]¶

Thrown when EVA is configured to run as a single instance, but it is already running according to Zookeeper.

exception eva.exceptions.ConfigurationException[source]¶

Base class for configuration related exceptions.

exception eva.exceptions.DuplicateEventException[source]¶

Thrown when trying to add an event to the event queue and it already exists.

exception eva.exceptions.EvaException[source]¶

Base exception for all exceptions thrown by EVA.

exception eva.exceptions.EventTimeoutException[source]¶

Thrown when the next event did not arrive in the expected time period.

exception eva.exceptions.GridEngineParseException[source]¶

Thrown when output parsing from GridEngine commands yields unexpected results.

exception eva.exceptions.InvalidConfigurationException[source]¶

Thrown when configuration does not make sense.

exception eva.exceptions.InvalidEventException[source]¶

Thrown when a received event is not valid for processing.

exception eva.exceptions.InvalidGroupIdException[source]¶

Thrown when a group_id is incompatible with Zookeeper.

exception eva.exceptions.InvalidRPCException[source]¶

Thrown when an RPC call contains invalid data.

exception eva.exceptions.JobNotCompleteException[source]¶

Thrown when an operation is performed on a Job object that is not complete, but should be. Treat this exception as a BUG!

exception eva.exceptions.JobNotGenerated[source]¶

Thrown by an adapter when a resource event fits the processing criteria, but does not need processing after all.

exception eva.exceptions.MissingConfigurationException[source]¶

Thrown when a configuration variable is missing.

exception eva.exceptions.MissingConfigurationSectionException[source]¶

Thrown when a configuration section is requested from a ConfigParser object, but does not exist.

exception eva.exceptions.RPCException[source]¶

Base class for RPC exceptions.

exception eva.exceptions.RPCFailedException[source]¶

Thrown when an RPC call fails.

exception eva.exceptions.RPCInvalidRegexException[source]¶

Thrown then the instance_id in a RPC message is not a valid regular expression.

exception eva.exceptions.RPCWrongInstanceIDException[source]¶

Thrown when the RPC message does not match our configured EVA instance ID.

exception eva.exceptions.ResourceTooOldException[source]¶

Thrown when an event is outdated because the Productstatus resource it refers to has been overwritten with more recent data.

exception eva.exceptions.RetryException[source]¶

Thrown when event processing cannot be completed due to a transient error on an underlying resource, typically a network or service outage.

exception eva.exceptions.ShutdownException[source]¶

Thrown when EVA is sent a SIGINT or SIGTERM signal.

exception eva.exceptions.ZooKeeperDataTooLargeException[source]¶

Thrown when trying to store a message in ZooKeeper that is too big.
